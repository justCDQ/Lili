# 后端与数据实践项目

## 1. 编程与算法练习集

使用 JavaScript 和 Go 实现基础数据结构、常用算法、文件统计和命令行 Todo，包含边界测试与复杂度说明。

## 2. Go CLI

子命令、配置、日志、并发、取消、测试和二进制发布。

## 3. 企业任务 API

用户、组织、项目、任务、评论、附件、RBAC、多租户、OpenAPI、PostgreSQL 和 Docker。

## 4. 文件中心

分片上传、断点续传、S3、CDN、权限、版本、搜索、异步处理和审计。

## 5. 分布式任务系统

API、Queue、Worker、Retry、DLQ、Idempotency、Heartbeat、Recovery 和 Metrics。

## 6. 数据分析系统

前端埋点、Event API、Queue、ClickHouse、Funnel、Retention 和 Dashboard。

## 7. 企业级综合平台

Go、PostgreSQL、Redis、MQ、S3、Search、Docker、Kubernetes、OpenTelemetry 和 CI/CD。
