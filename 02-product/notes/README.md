# 产品模块学习笔记

每完成一个路线图模块，就在这里沉淀产品概念、分析框架和实际案例。复制 [产品笔记模板](../notes-template.md)，避免只摘抄书籍或课程。

建议结构：

```text
notes/
├── foundations/
├── product-teardowns/
├── problem-evidence/
├── requirements-mvp/
├── metrics-experiments/
├── market-business/
└── b2b-ai-products/
```

每份笔记至少包含概念解释、适用/不适用条件、一个真实案例、反例、可验证证据和自己的判断。
