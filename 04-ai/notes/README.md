# AI 模块学习笔记

每完成一个路线图模块，就在这里沉淀相对稳定的原理和工程方法。复制 [AI 笔记模板](../notes-template.md)。某一天的模型对比和实验放在 `daily/`，经过复测仍成立的结论再提炼到这里。

建议结构：

```text
notes/
├── foundations/
├── model-api-prompt/
├── context-rag/
├── tools-mcp/
├── workflows-agents/
├── evaluation-observability/
└── safety-production/
```

每份笔记记录资料/规范版本和最后验证日期，区分稳定原理、当前实现、实验结论与尚未验证的判断。
