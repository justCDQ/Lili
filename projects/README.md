## 维护规则

- 每个综合项目有独立目录和 `README.md`。
- 项目主页记录目标、指标、里程碑、决策、证据和复盘链接。
- 代码可放在独立仓库；这里保留路线图、产品与工程证据的索引。
- 项目状态只使用：`idea`、`planned`、`active`、`paused`、`done`、`archived`。
