# 前端模块学习笔记

每完成一个路线图模块，就在这里沉淀一份能够独立阅读的笔记。复制 [前端笔记模板](../notes-template.md)，按主题组织，不按日期堆放。

建议结构：

```text
notes/
├── web-basics/
├── html-css/
├── javascript/
├── typescript-frameworks/
├── browser-performance/
├── architecture-tooling/
└── testing-security-observability/
```

文件名使用稳定主题，例如 `event-loop.md`、`css-stacking-context.md`。笔记应包含原理、最小实验、调试或测试证据、边界、取舍和关联项目。
