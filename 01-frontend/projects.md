# 前端实践项目

项目按难度递进。每个项目都应包含 README、运行方式、完成标准、关键取舍和学习复盘。

## 1. 语义化个人网站

只用 HTML 与 CSS 完成首页、文章、作品和联系表单；支持响应式、键盘、深色模式、基础 SEO 与公开部署。

## 2. 原生 JavaScript 应用

实现一个 Todo、天气或搜索应用，包含 DOM、表单、URL、真实 API、加载/错误/空状态、本地存储和基础测试。

## 3. TypeScript 框架应用

使用一种主流框架完成登录、路由、CRUD、表单、请求缓存、权限、错误边界、类型校验和自动化测试。

## 4. 可访问注册流程

原生表单、键盘操作、自动填充、错误关联、无 JS 降级、深色、RTL。

## 5. 性能实验室

制造长任务、布局抖动、泄漏和大资源；使用 DevTools 记录优化证据。

## 6. 企业级前端模板

Monorepo、Design System、Request、Auth、Permission、Monitoring、i18n、Feature Flag、Tests、CI/CD。

## 7. AI 工作区前端

Chat、Streaming、Artifact、Citation、Tool Call、Approval、Task Progress、Diff、History。
